== State

The app should be accessible, so where possible it is advisable to base states on the corresponding ARIA attributes.

.Example button
```tsx
    <Button
        key={section.id}
        variant="link"
        aria-current={activeSection === section.id ? 'step' : undefined}
        className="w-full justify-start aria-[current=step]:font-bold"
        onClick={() => scrollToSection(section.id)}
    >
```

Here the button state is based on the https://a11y-guidelines.orange.com/en/articles/using-aria-current-attribute/[`aria-current`] attribute.
