= PLANNING STACK TEMPLATE Frontend Documentation
:description: Developer documentation for the PLANNING STACK TEMPLATE frontend monorepo
:experimental:

== Overview

This repository is the default Tinker Stack starter for PLANNING STACK TEMPLATE.

The base workspace is intentionally empty enough to let a team start from first principles instead
of deleting a built-in demo.

== Modules

.Frontend structure
image::Prototyp-Repo.drawio.png[Repository structure]

[cols="1,4"]
|===
|Module |Description

|@repo/api
|Shared domain types and enums

|@repo/docs
|This documentation site

|@repo/frontend
|Reserved for the production frontend

|@repo/msw
|Starter-safe MSW package

|@repo/mocks
|Starter-safe mock-data package

|@repo/prototype
|Minimal React Router prototype shell

|@repo/ui
|Shared UI primitives and design-system components
|===

Optional examples can be generated into `examples/<name>/`. These examples are independent from the
base workspace and are not part of the default Turborepo build graph.

== Technology stack

The default stack includes:

* React with TypeScript
* React Router for the starter prototype
* Tailwind CSS and shadcn/ui-style primitives
* Turborepo for workspace orchestration
* MSW for local API mocking
* Antora for docs
