= Login Flow (Prototyp)
:experimental:

== Überblick

Der Prototyp enthält noch kein Login-System.
Implementiere eine Authentifizierung passend zu deinem Projekt, z.B.:

* Cookie-basierte Authentifizierung mit MSW Handlers
* JWT oder Session Tokens
* Integration mit einem IAM-System

== Nächste Schritte

1. Definiere Test-Benutzer und Rollen in `@repo/mocks`
2. Erstelle MSW Handler für Login/Logout Endpoints
3. Implementiere Auth-Prüfung in geschützten Handlers

TIP: Das optionale Example-Projekt (`examples/`) enthält eine vollständige Login-Implementierung mit Mock-Benutzern als Referenz.
