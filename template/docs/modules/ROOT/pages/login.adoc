= Login Flow (Prototype)
:experimental:

== Overview

The prototype does not include a login system yet.
Implement authentication that fits your project, for example:

* Cookie-based authentication with MSW handlers
* JWT or session tokens
* Integration with an IAM system

== Next steps

1. Define test users and roles in `@repo/mocks`
2. Create MSW handlers for login/logout endpoints
3. Implement auth checks in protected handlers

TIP: The optional example project (`examples/`) contains a complete login implementation with mock users as a reference.
